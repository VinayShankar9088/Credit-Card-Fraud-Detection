# 💳 Credit Card Fraud Detection Dashboard

An end-to-end Machine Learning project for detecting fraudulent credit card transactions using Logistic Regression and Streamlit.

---

## Features

- Bulk CSV Upload
- Fraud Prediction
- Fraud Probability
- Interactive Dashboard
- Pie Chart
- Histogram
- Download Prediction Report
- Responsive Streamlit UI

---

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

---

## Dataset

Kaggle Credit Card Fraud Detection Dataset

Features:

- Time
- V1–V28
- Amount

Target:

- Class (0 = Genuine, 1 = Fraud)

---

## Machine Learning Workflow

1. Data Cleaning
2. Train-Test Split
3. Feature Scaling
4. Logistic Regression
5. Model Evaluation
6. Model Deployment using Streamlit

---

## Installation

```bash
git clone https://github.com/VinayShankar9088/Credit-Card-Fraud-Detection

cd Credit-Card-Fraud-Detection

pip install -r requirements.txt

streamlit run app.py
```
